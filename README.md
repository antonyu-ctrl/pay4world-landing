# Pay4World Landing (Fresh Rebuild)

## Run

```bash
npm install
npm run dev
```

## What’s inside
- Next.js (App Router) + TypeScript + Tailwind
- Sticky Hero (shrinks on first scroll)
- Canvas 2D generative network background (auto-play story: seed → propagate → weave → secondary → ambient)

## Assets
Place your assets under `public/assets/`.
- `public/assets/logo.png`
- `public/assets/hero-ref.png` (fallback image if needed)
